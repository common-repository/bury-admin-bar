Hide the Admin Bar in WordPress

== Description ==

Hides the Admin Bar in WordPress.

== Installation ==

1. Upload `bury_admin_bar.php` to the `wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy!, there are no settings for this.
